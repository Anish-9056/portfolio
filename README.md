# Anish Kumar — Portfolio

🚀 **Live:** [https://anishkumar.dev](https://anishkumar.dev)

## About

Personal portfolio website built with Angular 19, showcasing my experience
as a Full Stack Developer specializing in enterprise web applications.

## Tech Stack

- **Framework:** Angular 19
- **UI:** Angular Material 3 + Custom Theming
- **Styling:** SCSS with CSS Custom Properties
- **Build:** Angular CLI + Webpack
- **Deployment:** GitHub Pages / Vercel

## Features

- 🎨 Dynamic light/dark theme support
- 📱 Fully responsive design
- ⚡ Optimized performance (90+ Lighthouse)
- ♿ Accessible (WCAG compliant)
- 🔄 Smooth scroll animations

## Sections

- Hero Banner
- About Me
- Work Experience
- Featured Projects
- Contact

## Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```
